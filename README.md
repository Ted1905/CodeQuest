# >_ CodeQuest Matrix v1.0

A specialized algorithmic evaluation and training platform designed for the **National Olympiad Team** and high-performance students at **Kupuev Academy**. 

## [ SYSTEM OVERVIEW ]
CodeQuest is a high-security, randomized testing environment built to sharpen the competitive programming skills of students preparing for national and international informatics olympiads. 

## [ CORE FEATURES ]
*   **Dynamic Randomization:** Questions and options are shuffled for every agent to ensure unique test instances.
*   **Anti-Cheat Protocol:** Includes deep-level security to prevent source-code inspection (F12), text copying, and unauthorized right-click actions.
*   **Real-time Matrix Sync:** Test results, including scores and completion times, are transmitted directly to a secure central database via Google Sheets API.
*   **Coach Dashboard:** A private, encrypted command center for the Lead CS Instructor to monitor agent performance and statistics.

## [ TECHNOLOGIES ]
*   **Frontend:** Vanilla JavaScript, Cyberpunk-themed CSS.
*   **Backend:** Google Apps Script / Google Sheets API.
*   **Security:** Multi-layer JavaScript Obfuscation.
